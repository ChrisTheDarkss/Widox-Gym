Git comando para conectarse a sus propias ramas(Haganlo solo una vez. Despues de cada inicio nuevo en su pc estara guardado la conexion remota):

git remote add origin https://github.com/ChrisTheDarkss/Widox-Gym



Luego usas (para clonar la carpeta de tu branch):

git clone https://github.com/ChrisTheDarkss/Widox-Gym (nombre de tu branch)

despues:

git checkout (nombre de tu rama)

En caso de no saber las ramas disponibles usa:

git branch -a

Entonces:
 
git init (hacia la carpeta clonada)

Una vez hecho eso empiezas a agregar contenido a la carpeta clonada

Despues agregas los datos de la carpeta:

git add (Nombre de lo agregado mas su formato)

Una vez a√±adido todo lo nuevo escribes el comando para enviarlo al proceso:

git commit -m "(ingrese info del archivo agregado)"

Despues de eso escribes:

Git push

Y esta agregado en tu rama :D
